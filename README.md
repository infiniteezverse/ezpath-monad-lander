# MONSKILLS EZ-Path Landing Page

Landing page for **monad.myezverse.xyz** — the DEX routing layer for Monad agents.

## Quick Deploy

This repo is configured for Vercel deployment with Cloudflare DNS pointing.

### Deploy Status
- **Deployed to:** Vercel
- **Custom domain:** https://monad.myezverse.xyz
- **DNS:** Cloudflare CNAME pointing to `cname.vercel.com`

### Local Development

```bash
# Simple static site, no build step needed
# Just open index.html in your browser or serve locally
python -m http.server 8000
# Visit http://localhost:8000
```

### Production

Vercel auto-deploys on push to `main` branch.
