The Problem

Single DEX Routing

Agents query one DEX at a time. Results in 1-5% slippage per trade. Billions in annual leakage to arbitrageurs.

Centralized Trust

Aggregators control order flow. Agents must trust Uniswap/1Inch. Single point of failure. Front-run risk.

Manual Coordination

No standard way for agents to discover each other. Market fragmentation. Complexity and latency.

The Solution

Parallel multi-venue DEX routing with agent-native discovery and X402 micro-payments.

10-Venue Parallel Routing: Race Uniswap V3, Curve, Balancer, 0x, Aerodrome, CoW, and more simultaneously. Get the best price in 177ms.
Decentralized Agent Discovery: Agents find each other via .well-known/agent.json and OpenAPI. No gatekeepers. MONSKILLS registry aggregates available pricing.
X402 Micro-Payments: EIP-3009 TransferWithAuthorization enables agent-to-agent pricing. No API keys. Fully cryptographic. On-chain settlement.

Agent Quick Start

Get the best DEX price in 3 steps.

Step 1: Probe the API (No Payment)

Call the endpoint without payment to learn the toll:

curl "https://api.myezverse.xyz/api/v1/quote?chain=base&sellToken=0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913&buyToken=0x4200000000000000000000000000000000000006&sellAmount=1000000"

Response: 402 Payment Required with pricing tiers (Basic $0.03, Resilient $0.10, Institutional $0.50)

Step 2: Sign X402 Payment (EIP-3009)

Create an EIP-3009 TransferWithAuthorization signed message:

Domain: USD Coin (0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913) To: 0x13dDE704389b1118B20d2BCc6D3Ace749600e2ad (toll address) Value: 30000 atomic USDC (0.03 USDC) ValidBefore: now + 300 seconds Nonce: random 32 bytes

Sign with your agent's private key. No private key stored. Fully cryptographic.

Step 3: Get Best Price (With Payment)

Retry with the signed X-Payment header:

curl "https://api.myezverse.xyz/api/v1/quote?chain=base&sellToken=...&buyToken=...&sellAmount=..." \ -H "X-Payment: <base64-encoded-eip3009-signature>"

Response: 200 OK with best price from 10 DEX venues (0x, ParaSwap, Aerodrome, Uniswap V3, Curve, Balancer, Uniswap V2, 1Inch, CoW, Synthetix)
